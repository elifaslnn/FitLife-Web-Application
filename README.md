# FitLife – Online Fitness Coaching Platform

## Project Overview

FitLife provides users with personalized workout programs, nutrition plans, progress tracking, messaging, and online training content to help them achieve their fitness goals. The system supports three different user roles: Admin, Trainer, and Client.

## User Roles and Permissions
### Admin

* View and manage all user accounts

* Activate or deactivate user accounts

* Manage user roles

### Trainer

* Edit personal profile information

* View assigned clients

* Create and update workout and nutrition plans for clients

* Communicate with assigned clients via messaging

* Can have a maximum of 5 clients

### Client

* View and edit personal profile

* Access workout and nutrition plans

* Communicate with the assigned trainer

* Add, view, and edit progress records (weight, BMI, body fat, etc.)

* View graphical progress reports

###  User Registration and Authentication

* Login with email and password

* Secure password storage

* "Forgot Password" functionality

* Profile photo upload and update
